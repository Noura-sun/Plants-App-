# OverView <a name="overview"></a>
This application **Planto** helps plant enthusiasts manage and keep track of their plant care routines effortlessly Specifically indoor plants. Users can add reminders for each plant, specifying care details like watering, and sunlight requirements. They can easily update plant information as their needs change and delete reminders when they are no longer necessary. Each day, the app provides a list of all the care tasks to be completed for each plant, allowing users to mark them as done after caring for a plant. Once all tasks for the day are completed, the app sends a confirmation, giving users a sense of accomplishment and ensuring no plant is left unattended.

## Features <a name="features"></a>
* Add Reminders: Set specific care reminders for each plant with details on watering and sunlight needs, and Its location inside the house .
* Update Plant Info: Easily edit and update plant details as they grow or require changes.
* Delete Reminders: Remove reminders for plants no longer in your collection.
* Daily Care Overview: View a daily list of all care tasks to be completed.
* Mark Tasks as Done: Mark reminders as completed after caring for each plant.
* Completion Confirmation: Receive a confirmation notification when all tasks are completed for the day.




## Goals <a name="goals"></a>

* Simplify Plant Care Management: Provide users with an intuitive and efficient way to manage the care routines of their plants, from watering and sunlight  schedules.
* Increase User Engagement: Keep users motivated to care for their plants daily by providing reminders and a sense of accomplishment through task completion confirmations.
* Personalized Care for Each Plant: Allow users to add specific care details and update them as their plants grow, creating a personalized experience for each plant in their collection.
* Reduce Forgetfulness and Plant Neglect: Help users remember important care tasks with daily reminders, reducing the chances of missing essential care steps for each plant.
* Foster a Positive Habit of Plant Care: Encourage users to build and maintain a healthy plant care routine, promoting responsibility and mindfulness toward their plants.
* User-Friendly Experience: Create an application that is easy to navigate and enjoyable to use, designed with an intuitive interface in Swift using Xcode, suitable for both beginners and experienced plant enthusiasts.

## Setup <a name="setup"></a>

To set up the project locally, follow these steps:

* **Clone the Repository:**
  ```bash
  git clone https://github.com/Noura-sun/Plants-App-.git
  ```


* **Open in Xcode:**
  - Open the .xcodeproj or .xcworkspace file in Xcode.
  
   
* **Run the Application:**
  - Select your target device (simulator or connected device) in Xcode.
  - Press Cmd + R to build and run the application.
